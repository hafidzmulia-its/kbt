# Valley of Blue Invitation

Standalone invitation landing page built as an original concept in a premium blue wedding style.

Open:

- Open [index.html](D:\Downloads\KBT\invitation-site\index.html) directly in a browser.
- Optional guest name query: `index.html?to=Mr%20Andi%20Saputra`

Main files:

- [index.html](D:\Downloads\KBT\invitation-site\index.html)
- [styles.css](D:\Downloads\KBT\invitation-site\styles.css)
- [script.js](D:\Downloads\KBT\invitation-site\script.js)

What to customize first:

- Couple names, family names, event date, and venue text in `index.html`
- Bank account details in `index.html`
- Countdown target date in `script.js`
- Colors, spacing, and visual treatment in `styles.css`

Production note:

- The RSVP form is a front-end prototype only. Connect it to WhatsApp, Google Sheets, Formspree, Supabase, or your own backend before using it live.
